Final Project: PCA-Based Residual Modeling and OU Mean-Reversion Strategy
=========================================================================

This project implements a complete statistical arbitrage pipeline on crypto assets:
(1) PCA factor construction, (2) regression-based residual extraction,
(3) Ornstein-Uhlenbeck (OU) parameter estimation, (4) s-score generation,
and (5) rule-based trading and backtesting.

The strategy follows the framework in Avellaneda & Lee (2010) and produces
all required outputs including eigenvectors, eigenportfolios, signals,
and performance figures.


Directory Structure
-------------------

src/
    data_loader.py        # Loads and preprocesses price data
    pca_engine.py         # Rolling PCA factor construction
    residual_model.py     # Regression residual extraction
    ou_estimator.py       # OU parameter estimation per token
    s_score.py            # S-score computation and signal rules
    strategy.py           # Position management
    backtester.py         # Strategy backtesting engine
    plotting.py           # All plotting functions
main.py                   # End-to-end pipeline driver

data/                         # not tracked in git (see Data section below)
    coin_all_prices_full.csv
    coin_universe_150K_40.csv

output/                       # not tracked in git (generated by main.py)
    eigenvector1.csv
    eigenvector2.csv
    signals.csv
    figures/             # All PNG graphs (PCA, strategy results, factor plots)

tests/
    test_ou_estimator.py
    test_s_score.py
    test_backtester.py


Data
----

The data/ directory is not tracked in this repository due to file size.
Place the following files into data/ before running:

    data/coin_all_prices_full.csv       # hourly OHLCV prices for all tokens
    data/coin_universe_150K_40.csv      # hourly top-40 token universe by market cap

Both files use 'startTime' as the timestamp column with hourly UTC frequency.


Installation
------------

    pip install -r requirements.txt


How to Run
----------

    python main.py

This will:
    - Load hourly crypto prices
    - Construct PCA factors using a 240-hour rolling window
    - Estimate residuals via ridge regression
    - Fit OU parameters for each token
    - Compute s-scores and generate long/short signals
    - Backtest the strategy over the testing window
    - Save all required outputs into the output/ directory


How to Run Tests
----------------

    python -m pytest tests/


Outputs

Notes
-----
Transaction costs and slippage are not modelled. All backtest results assume
zero fees and perfect execution at hourly close prices.
